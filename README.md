# sql-data-warehouse-project
Build a modern data warehouse project with postgresql including ETL processes ,data modelling and analytics

Project Overview

This project demonstrates the design and implementation of a modern Data Warehouse using the Medallion Architecture (Bronze → Silver → Gold). The objective was to transform raw source data into clean, business-ready analytical datasets that support reporting and decision-making.

The project was built using PostgreSQL and follows industry-standard ETL practices including data ingestion, data cleansing, transformation, dimensional modeling, and automated data loading through stored procedures.
